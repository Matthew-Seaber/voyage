# 🌐 Voyage

Plan your next trip away from start to finish with Voyage, the ultimate travel planning app!

## 🛠 How to access
Use the live demo at the bottom of the README
### OR
- Download the project files as a `.ZIP` or clone the repo using `git clone https://github.com/matthew-seaber/voyage.git` in your terminal
- Run `npm install` to install the required dependencies  
- Run `npm run dev` to start a local development server (default: `http://localhost:3000`)

## 🎨 Features
- Flight planner
- Hotel search
- Weather forecast
- Event finder
- Trip overview
- Travel guides
### NOTE
The APIs used in this project have limited free usage so if the live demo is not working, please try again later or run the project locally.
